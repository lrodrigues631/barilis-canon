# North-Central

Área organizacional para conteúdo canônico público situado na faixa geográfica
de transição entre o Norte e a Região Central.

Esta pasta existe para organização física do repositório. Ela não representa uma
quarta macro-região oficial de Barilis.

Use `realms/` para reinos e territórios políticos migrados no futuro. Use
`places/` para locais, marcos e assentamentos migrados no futuro.
